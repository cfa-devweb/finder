## INSTALL PROJECT

**npm install**

**composer install**

## DATABASE

Create a new Database named : ***Kinder*** in mysql or PhpMyAdmin

The database connection is :

DB_HOST= **127.0.0.1** \
DB_PORT= **3306** \
DB_DATABASE= **kinder** \
DB_USERNAME= **root** \
DB_PASSWORD= **root**

Run the migration with this command ***php artisan migrate:fresh***


## CSS/SCSS

To compile scss to css, run this command ***npm run watch***
